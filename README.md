# 💊 SafeMeds – Smart Medication Management System

SafeMeds is a web-based healthcare prototype designed to help caregivers manage and monitor patient medication schedules efficiently.

## 🚀 Features
- Add multiple medication schedules
- Set alert types (Audio, Visual, Both)
- Real-time alert display for upcoming doses
- Mark medication as Taken or Missed
- Automatic activity logging system
- Clean and responsive user interface

## 💡 Purpose
This project simulates a smart medication dispenser system that reduces the risk of missed or incorrect doses, especially for elderly patients or individuals managing multiple medications.

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📊 Key Concepts Demonstrated
- User interaction and form handling
- State management using JavaScript
- Dynamic DOM manipulation
- Healthcare system simulation
